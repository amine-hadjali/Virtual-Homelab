# Windows Server 2022 Installation
### Installation Type
Custom installation (manual configuration).
### Firmware Choice
UEFI
### Secure Boot
Disabled
### Edition Chosen
Windows Server 2022 Standard Evaluation (Desktop Experience)
## VMware Tools Installation (Windows Server and Clients)
### Steps:
    1. In VMware menu → VM → Install VMware Tools
    2. Inside Windows Server → run the setup
    3. Follow installation wizard → click Next →Typical → Install → Finish
    4. Reboot the VM to apply changes
